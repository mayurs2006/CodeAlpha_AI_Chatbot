# AI Chatbot 🤖

A simple AI Chatbot built using Python, Flask, HTML, and CSS.

## Features
- Interactive chatbot interface
- Multiple predefined responses
- Clean and responsive UI
- Built using Flask framework

## Technologies Used
- Python
- Flask
- HTML
- CSS

## Project Structure

CodeAlpha_AI_Chatbot
│
├── app.py
├── requirements.txt
├── static
│   └── style.css
└── templates
    └── index.html

## Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open in browser:

```text
http://127.0.0.1:5000
```
